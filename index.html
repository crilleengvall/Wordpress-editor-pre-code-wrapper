Welcome to Wordpress tinyMCE-pre-code-wrapper webpage

What is it?

This is a plugin that adds a button to the tinyMCE editor in WordPress. The button either adds a

or wraps the selected text in it.

This works well for developer blogs that use some kind of code markup highlighter. For example the WP Code highlight.js plugin
